# nmap-easy

## Description

Have you always dreamed of making nmap easier to use? Here is the solution: 

nmap- easy

## Getting Started

### Dependencies

macOS:
* brew (The macOS Homebrew)
* Python3
* nmap
* figlet
* XCode Developer Tools 

Linux Distribution: 
* Python3
* nmap
* figlet
* git

### Installing

```
git clone https://github.com/0n1cOn3/nmap-easy.git
cd nmap-easy
chmod +x nmap-cli.sh
```

### Executing program

```
sudo ./nmap-cli.sh
```
